## Functionalities of Movie Web App

• login

• registration

• adding a movie

• viewing movies

• updating movie

• deleting movie


## Tools which I used to build server side of this app 

• Java

• Spring Data Jpa (Hibernate)

• Spring Security

• MySQL

• Thymeleaf
